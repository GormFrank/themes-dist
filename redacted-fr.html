Background

Redacted text is a sub-section, often inline, of published content that are obfuscated and unreadable. For some case, the number of redacted character is included and visually represented.

Recommendation

Make the redacted text to be sementically understood as an image and apply text alternatives techniques to ensure conformance with the web accessibility standard.

This technique leverage WAI-ARIA.

Technical specification

  • Use always the same character for obfuscation in your pages.

  • We recommend to use the Dark shade as your obfuscated character.

    • Name: Dark shade
    • Symbol: ▓
    • HTML Entity: &#9619;
    • Unicode: U+2593

    The dark shade character is a good fit when compared with the full block character because it makes the redacted text less prominent and make the surrounding text more readable.

  • Make the redacted text need to be in its own container. like: <span></span>
  • Add the attribute role="img" to the redacted text container.
  • Add the attribute aria-label="Redacted text" and modify the label accordingly to your context of use.
  • Add CSS class .redacted for an inline redacted text
  • If there no requirement to specify the number of redacted character, use the same amount of dark shade character for each redacted section and you will be able to reuse the same text label
  • If you need to specify the number of redacted character, add the dark shade character inside the container to be the same as the number of redacted character. Also, adjust the text label accordingly.
