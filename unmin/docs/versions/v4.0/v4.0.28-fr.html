Quoi de neuf?

Version 4.0.28.1

  • Réparation d'un problème qui empêchait la fonctionnalité BOEW pour effectuer la validation de formulaire de démarrer.

Version 4.0.28

  • Mineur - Changement dans le balisage des images SVG
    • Décision de conception 1 - Utilidation des SVG avec l'élément img plutôt que l'élément object (en anglais)
  • Mineur - Datepicker polyfill - La date d'initialisation a été ajusté lorsque la date minimale est plus grande que la date actuelle
  • Mineur - Graphique - Ajout d'étiquettes pour les graphiques sectoriel
  • Développeur de la BOEW - Majeur - Changement dans le script de compilation
    • Les développeur de la BOEW devrons peut-être supprimer le cache de npm ainsi que les répertoires "node_modules" et "lib" avant d'exécuter "script/setup" pour être capable de compiler avec succès.

Canada.ca (GCWeb):

  • Mineur - Changement dans le balisage des images SVG
    • Décision de conception 1 - Utilidation des SVG avec l'élément img plutôt que l'élément object (en anglais)
  • Mineur - Section en vedette - Ajout d'un exemple avec un item
  • Mineur - Section en vedette - Changement de balisage
  • Mineur - Les priorités du GC a été enlevé
  • Mineur - Amélioration du balisage pour l'image de marque d'oganisme indépendant
  • Mineur - Mise à jour du méga menu
  • Information - Ajout d'un page pour la mise à l'essaie de balisage maintenant déconseillé
  • Développeur de la BOEW - Majeur - Changement dans le script de compilation
    • Les développeur de la BOEW devrons peut-être supprimer le cache de npm ainsi que les répertoires "node_modules" et "lib" avant d'exécuter "script/setup" pour être capable de compiler avec succès.

Les thèmes suivant ont été abandonné:

  • Base
  • Facilité d’emploi Web du gouvernement du Canada
  • Gouvernement du Canada pour les sites intranet
  • Plate-forme de gouvernement ouvert (PGO)

Si vous estes intéressé d'avoir un de ces thèmes ci-dessus publié avec la version 4.0.28 de la BOEW, veuillez joindre la discussion sur Github via le billet #8366.
