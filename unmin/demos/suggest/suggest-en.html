Datalist JSON suggestions

  • Documentation
  • Questions or comments?

Load suggestions from an array of string in a JSON file.

<form action="#">
	<div class="form-group">
		<label for="input-id-1">Surveys</label>
		<input id="input-id-1" list="datalist-id-1" />
	</div>
	<datalist id="datalist-id-1" data-wb-suggest="demo/suggest-en.json#/suggestions">
		<template>
			<option></option>
		</template>
	</datalist>
</form>

Limiting the number of displayed results

The following example limit the number of result to four items. The HTML5 specification recommends to limits between four to seven items.

Note: Using the same datalist for two different input may conflict when limiting the number of results.

Source code 
<form action="#">
	<div class="form-group">
		<label for="input-id-2">Surveys</label>
		<input id="input-id-2" list="datalist-id-2" />
	</div>
	<datalist id="datalist-id-2" data-wb-suggest="demo/suggest-en.json#/suggestions" data-wb-limit="4">
		<template>
			<option></option>
		</template>
	</datalist>
</form>

Apply diferent type of filter

The autocomplete implementation vary from browser. IE trend to show start of matching word, Chrome show any item that contains the string, Firefox also show any item that contains the string but limit the number of displayed result to five items.

Note: Using the same datalist for two different input may conflict when limiting the number of results.

Using the filter type work best when combined with data-wb-limit options

Filter type any (default)

Source code 
<form action="#">
	<div class="form-group">
		<label for="input-id-3">Surveys</label>
		<input id="input-id-3" list="datalist-id-3" />
	</div>
	<datalist id="datalist-id-3" data-wb-suggest="demo/suggest-en.json#/suggestions" data-wb-filter-type="any">
		<template>
			<option></option>
		</template>
	</datalist>
</form>

Filter type startWith

Source code 
<form action="#">
	<div class="form-group">
		<label for="input-id-4">Surveys</label>
		<input id="input-id-4" list="datalist-id-4" />
	</div>
	<datalist id="datalist-id-4" data-wb-suggest="demo/suggest-en.json#/suggestions" data-wb-filter-type="startWith">
		<template>
			<option></option>
		</template>
	</datalist>
</form>

Filter type word

Source code 
<form action="#">
	<div class="form-group">
		<label for="input-id-5">Surveys</label>
		<input id="input-id-5" list="datalist-id-5" />
	</div>
	<datalist id="datalist-id-5" data-wb-suggest="demo/suggest-en.json#/suggestions" data-wb-filter-type="word">
		<template>
			<option></option>
		</template>
	</datalist>
</form>
