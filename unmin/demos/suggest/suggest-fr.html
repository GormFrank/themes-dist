Suggestion pour datalist via JSON

  • Documentation
  • Questions ou commentaires?

Charge des suggestions d'un tableau de chaîne de caractère à partir d'un fichier JSON.

<form action="#">
	<div class="form-group">
		<label for="input-id-1">Enquête</label>
		<input id="input-id-1" list="datalist-id-1" />
	</div>
	<datalist id="datalist-id-1" data-wb-suggest="demo/suggest-fr.json#/suggestions">
		<template>
			<option></option>
		</template>
	</datalist>
</form>

Limiter le nombre de résultat affiché

L'example suivant limite le nombre de résultat à quatre items. La spécification du HTML5 recommande de limiter les résultats entre quatre et sept items.

Note: L'utilisation d'un même datalist pour plusieurs input pourrait créer des conflits lorsque le nombre de résultat est limité.

Code source 
<form action="#">
	<div class="form-group">
		<label for="input-id-2">Enquête</label>
		<input id="input-id-2" list="datalist-id-2" />
	</div>
	<datalist id="datalist-id-2" data-wb-suggest="demo/suggest-fr.json#/suggestions" data-wb-limit="4">
		<template>
			<option></option>
		</template>
	</datalist>
</form>

Appliquer différent type de filtre

L'implémentation de la fonctionalité de saisie semi-automatique est légèrement différente d'un fureteur à un autre. IE a tendance à afficher seulement les résultats qui débute part, Chrome affiche tout les résultats qui est contenu dans la chaine de charactère, Firefox agis similairement à Chrome avec l'exception qu'il limite l'affichage à cinq résultats.

Note: L'utilisation d'un même datalist pour plusieurs input pourrait créer des conflits lorsque le nombre de résultat est limité.

L'utilisation du type de filtre est plus efficace lorsqu'il est utilisé conjointement avec l'option data-wb-limit.

Type de filtre: n'importe où (any) [défault]

Code source 
<form action="#">
	<div class="form-group">
		<label for="input-id-3">Enquête</label>
		<input id="input-id-3" list="datalist-id-3" />
	</div>
	<datalist id="datalist-id-3" data-wb-suggest="demo/suggest-fr.json#/suggestions" data-wb-filter-type="any">
		<template>
			<option></option>
		</template>
	</datalist>
</form>

Type de filtre: débute par (startWith)

Code source 
<form action="#">
	<div class="form-group">
		<label for="input-id-4">Enquête</label>
		<input id="input-id-4" list="datalist-id-4" />
	</div>
	<datalist id="datalist-id-4" data-wb-suggest="demo/suggest-fr.json#/suggestions" data-wb-filter-type="startWith">
		<template>
			<option></option>
		</template>
	</datalist>
</form>

Type de filtre: mot (word)

Code source 
<form action="#">
	<div class="form-group">
		<label for="input-id-5">Enquête</label>
		<input id="input-id-5" list="datalist-id-5" />
	</div>
	<datalist id="datalist-id-5" data-wb-suggest="demo/suggest-fr.json#/suggestions" data-wb-filter-type="word">

			<option></option>
		</template>
	</datalist>
</form>
